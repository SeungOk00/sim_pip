# RFdiffusion 설정 키 설명

대상 파일:
- `C:\project\sim_pip\RFdiffusion\config\inference\base.yaml`
- `C:\project\sim_pip\RFdiffusion\config\inference\symmetry.yaml`

## 1) base.yaml

### inference
- `inference.input_pdb`: 입력 PDB 경로(조건부 설계 시 사용, `null`이면 없음)
- `inference.num_designs`: 생성할 디자인 개수
- `inference.design_startnum`: 출력 파일 번호 시작값
- `inference.ckpt_override_path`: 기본 대신 사용할 체크포인트 경로
- `inference.symmetry`: 대칭 모드(`c2`, `d3`, `tetrahedral` 등)
- `inference.recenter`: 좌표 재중심화 여부
- `inference.radius`: 초기/배치 반경 관련 스케일
- `inference.model_only_neighbors`: 대칭체에서 이웃 서브유닛만 모델링할지
- `inference.output_prefix`: 출력 파일 prefix
- `inference.write_trajectory`: 중간 확산 trajectory 저장 여부
- `inference.scaffold_guided`: scaffold-guided 모드 활성화 여부
- `inference.model_runner`: 사용할 샘플러/러너 타입
- `inference.cautious`: 출력 파일 이미 있으면 스킵 여부
- `inference.align_motif`: motif 정렬 수행 여부
- `inference.symmetric_self_cond`: 대칭 self-conditioning 사용 여부
- `inference.final_step`: 역확산 종료 스텝
- `inference.deterministic`: 시드 고정(재현성 모드) 여부
- `inference.trb_save_ckpt_path`: `.trb`에 기록할 체크포인트 경로
- `inference.schedule_directory_path`: 스케줄 파일 디렉터리 override
- `inference.model_directory_path`: 모델 디렉터리 override
- `inference.cyclic`: cyclic peptide 관련 모드
- `inference.cyc_chains`: cyclic 처리할 체인 지정

### contigmap
- `contigmap.contigs`: contig 문자열(길이/모티프/체인 브레이크 정의)
- `contigmap.inpaint_seq`: 서열 마스킹할 residue 범위
- `contigmap.inpaint_str`: 구조 마스킹할 residue 범위
- `contigmap.inpaint_str_helix`: helix로 강제할 범위
- `contigmap.inpaint_str_strand`: strand로 강제할 범위
- `contigmap.inpaint_str_loop`: loop로 강제할 범위
- `contigmap.provide_seq`: 고정/제공할 서열 범위
- `contigmap.length`: 전체 길이 제약 범위

### model
- `model.n_extra_block`: extra block 수
- `model.n_main_block`: main block 수
- `model.n_ref_block`: refinement block 수
- `model.d_msa`: MSA embedding 차원
- `model.d_msa_full`: full MSA 차원
- `model.d_pair`: pair feature 차원
- `model.d_templ`: template feature 차원
- `model.n_head_msa`: MSA attention head 수
- `model.n_head_pair`: pair attention head 수
- `model.n_head_templ`: template attention head 수
- `model.d_hidden`: hidden 차원
- `model.d_hidden_templ`: template hidden 차원
- `model.p_drop`: dropout 비율
- `model.SE3_param_full.*`: full graph용 SE3 transformer 파라미터
- `model.SE3_param_topk.*`: top-k graph용 SE3 transformer 파라미터
- `model.freeze_track_motif`: motif track 업데이트 고정 여부
- `model.use_motif_timestep`: motif 전용 timestep embedding 사용 여부

### diffuser
- `diffuser.T`: 확산 총 timestep 수
- `diffuser.b_0`: beta 시작값
- `diffuser.b_T`: beta 끝값
- `diffuser.schedule_type`: 노이즈 스케줄 타입(예: linear)
- `diffuser.so3_type`: 회전 확산 방식(예: igso3)
- `diffuser.crd_scale`: 좌표 스케일 계수
- `diffuser.partial_T`: partial diffusion 시작 timestep
- `diffuser.so3_schedule_type`: SO(3) 스케줄 타입
- `diffuser.min_b`: SO(3) 스케줄 최소 beta
- `diffuser.max_b`: SO(3) 스케줄 최대 beta
- `diffuser.min_sigma`: SO(3) 최소 sigma
- `diffuser.max_sigma`: SO(3) 최대 sigma

### denoiser
- `denoiser.noise_scale_ca`: CA 좌표 노이즈 스케일
- `denoiser.final_noise_scale_ca`: 마지막 스텝 CA 노이즈 스케일
- `denoiser.ca_noise_schedule_type`: CA 노이즈 스케줄 타입
- `denoiser.noise_scale_frame`: frame 노이즈 스케일
- `denoiser.final_noise_scale_frame`: 마지막 스텝 frame 노이즈 스케일
- `denoiser.frame_noise_schedule_type`: frame 노이즈 스케줄 타입

### ppi
- `ppi.hotspot_res`: binder 설계에서 hotspot residue 지정

### potentials
- `potentials.guiding_potentials`: 사용할 보조 potential 목록
- `potentials.guide_scale`: potential 가중치 스케일
- `potentials.guide_decay`: guide decay 방식
- `potentials.olig_inter_all`: 올리고머 inter-chain contact potential 일괄 적용
- `potentials.olig_intra_all`: 올리고머 intra-chain contact potential 일괄 적용
- `potentials.olig_custom_contact`: 체인쌍별 custom contact 문자열
- `potentials.substrate`: substrate 관련 potential 입력

### contig_settings
- `contig_settings.ref_idx`: 참조 인덱스 직접 지정
- `contig_settings.hal_idx`: 출력(hal) 인덱스 직접 지정
- `contig_settings.idx_rf`: RF 인덱스 직접 지정
- `contig_settings.inpaint_seq_tensor`: 서열 inpaint 텐서 직접 지정

### preprocess
- `preprocess.sidechain_input`: 입력에 sidechain 포함 여부
- `preprocess.motif_sidechain_input`: motif sidechain 입력 포함 여부
- `preprocess.d_t1d`: 1D template feature 차원
- `preprocess.d_t2d`: 2D template feature 차원
- `preprocess.prob_self_cond`: self-conditioning 확률
- `preprocess.str_self_cond`: 구조 self-conditioning 사용 여부
- `preprocess.predict_previous`: 이전 step 예측값 사용 여부

### logging
- `logging.inputs`: 모델 입력 로깅 활성화 여부

### scaffoldguided
- `scaffoldguided.scaffoldguided`: scaffold-guided 모드 토글
- `scaffoldguided.target_pdb`: target PDB 사용 여부
- `scaffoldguided.target_path`: target 파일/폴더 경로
- `scaffoldguided.scaffold_list`: scaffold 목록 파일
- `scaffoldguided.scaffold_dir`: scaffold 디렉터리
- `scaffoldguided.sampled_insertion`: 삽입 길이 샘플링 값
- `scaffoldguided.sampled_N`: N-말단 샘플링 길이
- `scaffoldguided.sampled_C`: C-말단 샘플링 길이
- `scaffoldguided.ss_mask`: secondary structure 마스크 강도/범위
- `scaffoldguided.systematic`: systematic 스캔 모드
- `scaffoldguided.target_ss`: target secondary structure 특징 파일
- `scaffoldguided.target_adj`: target adjacency 특징 파일
- `scaffoldguided.mask_loops`: loop 마스킹 여부
- `scaffoldguided.contig_crop`: contig crop 설정

## 2) symmetry.yaml

- `defaults: - base`: `base.yaml`을 기본으로 불러오고 일부 값만 덮어씀

### inference
- `inference.symmetry`: 사용할 대칭 타입(기본 `c2`)
- `inference.model_only_neighbors`: 메모리 절약용 이웃 서브유닛만 모델링 여부
- `inference.output_prefix`: 대칭 샘플 출력 prefix (`samples/c2`)

### contigmap
- `contigmap.contigs`: 대칭 생성 시 길이 설정(예: `['100']`)
  - 대칭 체인 수로 나누어떨어지는 길이를 지정해야 함
